## Markdown 转 PDF（Android 面试题）
- 使用 md-to-pdf-cjk skill 将 `Android 高频面试题（100题）.md` 转为 PDF
- 原始 skill 脚本不支持代码块、引用块、HTML锚点等，编写增强版脚本 `/tmp/md_to_pdf_enhanced.py`
- 增强：macOS字体检测(Hiragino Sans GB)、代码块灰底等宽字体、引用块样式、链接转纯文本、表格Paragraph换行、页码
- 结果：71页 PDF，356KB，输出到项目根目录
- 注意：skill 文档写的是3参数(含title)但实际脚本只接受2参数
