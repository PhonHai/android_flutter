### 用户困惑解答
1. Flutter 代码在 `flutter_module/lib/` 里面（不在根目录），9 个文件 618 行
2. `.android/plugins_build_output/sqflite_android/` 是 sqflite 插件的编译产物（自动生成，不要改），就像 Room 的 aar 内部代码一样
3. AS 打开 native-android/ 看不到 Flutter 源码（只有 AAR 编译壳），需要单独打开 flutter_module/
4. Flutter 开发用 `flutter attach` 连到运行中的 App，热重载秒级生效
